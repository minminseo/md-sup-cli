
# コマンド一覧
```
msc link [<ファイルパス>]             タイトル付きリンクのテンプレ生成

msc details [<ファイルパス>]          詳細折りたたみのテンプレ生成

msc snip [<言語>] [<ファイルパス>]    コードスニペットを追加

msc table [<ファイルパス>]            テーブルのテンプレ生成

msc all [<ファイルパス>]              全テンプレ生成

msc version                           バージョン表示

msc help                              このヘルプを表示
```
